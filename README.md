# ByteTea
 Plugins bases on ASM


develop base on [ByteX](https://github.com/bytedance/ByteX)

## Usage
byteTeaVersion: 1.5

```gradle
repositories {
        maven { url 'https://dl.bintray.com/mikaelzero/maven'}
    }

 dependencies {
        classpath "com.bytedance.android.byteX:base-plugin:0.1.8"
     }

```
